# Contributing

Thanks for your interest in contributing to HomeKitKnock-S3.

## Quick Start
- Read `docs/esp32-s3-doorbell-architecture.md` for project context.
- Check `platformio.ini` for board and build flags.
- Use PlatformIO tasks or `pio` CLI for build/upload/monitor.

## Making Changes
- Keep changes small and focused.
- Document wiring or pin changes in the architecture doc.
- If you touch firmware behavior, note how to test it.

## Pull Requests
- Describe the problem and the solution.
- Include test notes (build, upload, or serial monitor steps).
